---- Simón Dice ----

Este proyecto es un juego de memoria interactivo desarrollado como una aplicación de escritorio. 
El objetivo es desafiar la capacidad de retención del usuario mediante secuencias de palabras aleatorias 
que deben ser ingresadas bajo presión de tiempo.

---- ¿Cómo funciona? ----

1. Se muestran palabras aleatorias en pantalla
2. El usuario debe escribirlas correctamente
3. Cuenta con un temporizador de 30 segundos
4. Se evalúa el desempeño del jugador

---- Características ----

⏱️ Temporizador en tiempo real
🔤 Generación de palabras aleatorias
💾 Guardado de puntajes
🎯 Interfaz gráfica en Java
⚡ Lógica estructurada por capas

---- Tecnologías utilizadas ----
- Java
- Programación Orientada a Objetos (POO)
- Swing (Interfaz gráfica)
